# Day 09 Basic Prediction Models — 시험 대비 요약

## 1. 선형 회귀

### 개념
- 입력변수와 **실수형** 출력변수 간 수학적 관계를 추정 → 신규 $X$로 $Y$ 예측 (**가상계측** 등)
- **단순선형회귀**: $Y=\beta_0+\beta_1 X+\varepsilon$, 추세선 $f(X)=\beta_0+\beta_1 X$
- 학습 추정식: $\hat{Y}=\hat{f}(X)=\hat{\beta}_0+\hat{\beta}_1 X$
- $\hat{\beta}_1$: $X$ 한 단위 증가 시 $Y$ 변화량 (= 입력변수 **중요도**)

### 잔차·OLS
- **잔차**: $e_i=y_i-\hat{y}_i$
- **SSE** $=\sum e_i^2$ 를 최소화 → **최소자승법 (OLS)**
- $\hat{\beta}_1=\dfrac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sum(x_i-\bar{x})^2}$, $\hat{\beta}_0=\bar{y}-\hat{\beta}_1\bar{x}$

### 성능 지표 $R^2$
- $y_i-\bar{y}=(y_i-\hat{y}_i)+(\hat{y}_i-\bar{y})$ → **SST = SSR + SSE**
- $R^2=\dfrac{SSR}{SST}=1-\dfrac{SSE}{SST}$ (범위 보통 0~1, **1에 가까울수록 설명력↑**)
- 의미: $\bar{Y}$로 예측할 때보다 회귀가 잔차를 얼마나 줄였는가
- **주의**: 학습 $R^2$ 최소는 0 / **검증·비선형**에서는 $R^2<0$ 가능 (평균보다 못함·과적합 등)
- $Y$가 평균 근처에만 모이면 $R^2\approx 0$이어도 정상일 수 있음 → **MSE**로 보완 판단

### 다중선형회귀
- $\hat{Y}=\hat{\beta}_0+\hat{\beta}_1 X_1+\cdots+\hat{\beta}_p X_p$
- $\hat{\beta}_i$: 다른 변수 고정 시 $X_i$ 한 단위 증가에 따른 $Y$ 변화량
- OLS 행렬해: $\hat{\boldsymbol{\beta}}=(X'X)^{-1}X'y$
- 변수↑ → 예측력↑ 가능하나 **과적합·다중공선성** 위험

### 범주형·교호작용
- **One-Hot Encoding**: 범주 항목마다 0/1 변수 생성. 구현 쉬움 / 항목 많으면 변수 폭증·과적합
- **교호작용** ≠ 다중공선성
  - 다중공선성: $X$들이 $Y$ 설명에 **중복**
  - 교호작용: $X$들의 **상호작용**이 $Y$를 추가로 설명 → $\hat{Y}=\cdots+\hat{\beta}_3 X_1 X_2$

## 2. 계수 축소 회귀 (Shrinkage)

### 개요
- 목적식: Minimize $SSE+f(\hat{\beta})$ — 잔차 + **회귀계수 축소(통제항)**
- 영향 없는 계수를 0(또는 0 근처)으로 → **과적합·다중공선성** 완화
- $\lambda$: **하이퍼 파라미터**. 클수록 축소↑ (너무 크면 **과소적합**)

| 방법 | 통제항 | 특징 |
|------|--------|------|
| **Ridge** | $\lambda\sum\hat{\beta}_j^2$ | 계수를 **0에 가깝게**. 완전 0은 거의 불가 |
| **Lasso** | $\lambda\sum\|\hat{\beta}_j\|$ | 계수를 **완전 0**으로 → **변수 선택**에 유리 |

### 선택 가이드
- 변수들이 $Y$에 **비슷하게** 영향 → **Ridge** 유리
- 영향력 **편차 큼** / 소수 중요 변수만 남기고 싶음 → **Lasso** 유리
- $\lambda$는 검증 **MSE 최소** 지점으로 탐색
- 제약형: Ridge $\sum\hat{\beta}_j^2\le s$ / Lasso $\sum|\hat{\beta}_j|\le s$ ($s\propto 1/\lambda$)

## 3. 로지스틱 회귀

### 개념
- 출력 = **범주**(정상/불량 등). **이진 분류**·FDC에 사용
- 목표: 불량 확률 $p=P(\text{불량}|X)$ 예측. 보통 $p\ge 0.5$면 불량 (**Threshold**는 사용자 조정 가능)
- 선형회귀는 예측이 $[-\infty,+\infty]$로 나와 확률 해석 곤란 → **로지스틱 곡선** 사용

### 핵심 식
- **오즈(Odds)**: $\dfrac{p}{1-p}$ / **로짓(Logit)**: $\ln\dfrac{p}{1-p}=\hat{\beta}_0+\hat{\beta}_1 X_1+\cdots$
- 확률: $p(X)=\dfrac{e^{\hat{\beta}_0+\cdots}}{1+e^{\hat{\beta}_0+\cdots}}$
- $\hat{\beta}_k>0$: $X_k$↑ → 불량($Y=1$) 확률↑ / $\hat{\beta}_k<0$: 정상 확률↑

### 학습
- OLS 아님 → **최대우도법 (Maximum Likelihood)**
- 손실함수: **Binary Cross Entropy**  
  $-\sum[y_i\ln p(x_i)+(1-y_i)\ln(1-p(x_i))]$ 최소화
- Class Imbalance 시 **Threshold** 조정으로 Recall·Precision 등 트레이드오프 조절 가능

## 4. 의사결정나무 (CART)

### 구조·원리
- **의사결정 규칙**을 나무로 표현. 분류·회귀 모두 가능
- 구성: **Root / Intermediate / Leaf**, **Branch**
- 입력공간을 **축에 평행**(수직·수평)하게 분할 → 영역 내 **클래스 동질성** 최대화
- 대표 알고리즘: **CART** (**이진 분할**). ID3/C4.5는 다중 분할

### 지니 불순도·정보 이득
- **Gini**: $Gini(S)=1-\sum p_i^2$  
  - 이진: $1-p_+^2-p_-^2$  
  - 0 = 순수(한 클래스만) / $p_+=0.5$일 때 최대(혼잡)
  - 의미: 임의 2개 샘플의 클래스가 다를 확률
- **Information Gain**:  
  $IG(S,A)=Gini(S)-\sum\dfrac{|S_v|}{|S|}Gini(S_v)$  
  → 분할 전후 혼잡도 감소량. **IG 최대**인 변수·경계를 선택해 재귀 분할
- 연속형: 정렬 후 클래스 변화 지점 후보 → **Binary Discretization**으로 최적 Threshold

### 가지치기 (과적합 방지)
| 종류 | 내용 |
|------|------|
| **사전** | Full Tree 전 중단 (`max_depth`, `max_leaf_nodes`, `min_samples_leaf`, `min_samples_split`) |
| **사후** | Full Tree 후 subtree 제거. CART는 **Cost-Complexity Pruning** |

- $CC(T)=Err(T)+\alpha\cdot|T|$  
  - $Err$: 검증 오분류율 / $|T|$: 끝 노드 수 / $\alpha$↑ → 더 단순한 나무

### 회귀 나무
- 분할 기준: Gini 대신 **분산 감소**(Variance Difference)
- 예측값: Leaf에 속한 학습 데이터의 **평균**

### 장단점
- 장점: **해석 용이**, 연속·범주형 모두 처리
- 단점: 복잡한 관계에 약함, **과적합** 쉬움 → 실무에선 **앙상블**과 병행

### 참(O) / 거짓(X)

1. $\hat{\beta}_1$은 $X$가 한 단위 증가할 때 $Y$의 변화량을 의미함.  
   **O**

2. 학습 데이터에 대한 선형회귀 $R^2$는 음수가 될 수 있음.  
   **X** (학습 $R^2$ 최소는 0. 음수는 주로 검증·비선형에서 발생)

3. 교호작용과 다중공선성은 같은 개념임.  
   **X** (교호작용=상호작용의 추가 설명력 / 다중공선성=$X$간 중복)

4. Ridge는 계수를 완전 0으로 만들어 변수 선택에 최적임.  
   **X** (완전 0은 Lasso 쪽. Ridge는 0에 가깝게만 축소)

5. Lasso에서 $\lambda$를 키우면 항상 다중선형회귀보다 검증 MSE가 작아짐.  
   **X** ($\lambda$가 과도하면 성능이 오히려 나빠질 수 있음)

6. 선형회귀로 정상/불량(0/1)을 바로 적합하면 예측이 음수·1 초과로 나와 해석이 어려움.  
   **O**

7. 로지스틱에서 $\hat{\beta}_k$가 음수이면 $X_k$ 증가 시 불량 확률이 높아짐.  
   **X** (음수면 불량 확률↓, 정상 쪽↑)

8. 지니 불순도 0은 해당 영역에 한 클래스만 있음을 의미함.  
   **O**

9. CART는 노드 분할 시 다중 분할(Multi-Way Split)을 기본으로 함.  
   **X** (CART는 이진 분할. 다중 분할은 ID3/C4.5)

10. 사전 가지치기는 Full Tree를 만든 뒤 subtree를 제거하는 방법임.  
    **X** (그건 사후 가지치기. 사전은 완성 전 중단)

11. Cost-Complexity에서 $\alpha$가 커질수록 더 복잡한 나무가 선호됨.  
    **X** ($\alpha$↑ → 복잡도 패널티↑ → 더 간결한 나무)

12. 의사결정나무는 해석이 쉽지만 과적합이 잘 발생함.  
    **O**
