## 1. 시계열 데이터 분석
**시계열 데이터 분석 실패 원인 2가지**
	- 순서를 반영 안해줘서
	- 맞춰야하는 대상이 Y인지 X인지 혼란한 경우

> 평가 지표들의 의미를 잘 알고 모델의 성능을 비교할 것!

**ANN**은 전체 데이터를 활용하여 가중치 업데이트함.
**RNN(순환신경망)** 은 전체 가중치는 고정, 순차적으로 관측치를 활용하여 시점 별 정보가 다음 관측치에 반영되도록 은닉층을 계산한다.
	ANN보다 더 많은 parameter가 발생한다.
	직전 data의 은닉층 정보를 다음 data에 반영할 때, 최근의 data가 더 많이 반영된다. ==시퀀스가 길어질수록 과거 정보는 소멸된다.==(기울기 소실 문제)

**LSTM**은 RNN의 단점을 해결하기 위해 등장했다. 
	Long Shor-Term Memory

RNN에는 여러 모델이 있다. 기본 모형은 *'바닐라'* 라 부른다.
	GRU, Bi-directinal LSTM(순/역방향 분석 결과를 mix)가 RNN에 속함

## 2. 센서 데이터 분석
현업에 적용 가능성 높은 내용

**FDC data**를 분석하는 방법들
	1) 다양한 feature 변환 방식(통계량)을 통해 변수들로 변환할 수 있다. 정형 data로 변환하고나면 ML, MLP, LSTM 등으로 모델링 가능하다.
	2) 데이터를 특정 길이로 잘라서 합친 후 2D배열로 변환할 수도 있다. CNN, LSTM으로 모델링 가능하다.
		잘라진 경계의 data는 소실되는 단점이 있다.
	3) 시계열 데이터를 이미지화(인코딩)하면 딥러닝 모델에 적용하여 분류 모형이나 이상 탐지 알고리즘에 적용할 수 있다. CNN, LSTM으로 모델링 가능하다.

**합성곱 신경망**(CNN:Convolution Neural Network)은 인간의 시각 처리 방식을 모방하여 주로 이미지나 영상 데이터를 분석하는 데 사용되는 딥러닝 모델입니다. 입력 데이터의 특징을 스스로 추출하는 합성곱층(Convolutional Layer)과 이미지의 크기를 줄여 연산량을 낮추는 풀링층(Pooling Layer)을 조합하여 객체 인식 및 분류에 뛰어난 성능을 발휘합니다.

## 3. WDCNN (Wide Deep CNN)
변환으로 인한 정보 손실 없이 원시 진동 신호 그대로 CNN에 넣는 방법.
==첫 단계에는 넓은 커널==(필터)을 적용해서 노이즈를 제거해서 중요한 특징을 추출한다. 이후에 작은 커널로 세밀한 특징을 추출한다.
도메인 이동(환경 변화)시 성능이 떨어지는데...
	**AdaBN**(Adaptive Batch Normalization)이 해결책이다. 모델 전체를 재학습하지 않고 정규화 파라미터를 새로운 도메인 데이터로 업데이트한다.

## 4. 시계열 Foundation Model
대량의 시계열 데이터로 사전 학습이 완료된 모델. 내 데이터를 학습시키지 않고 ==Zero-shot으로 이상 탐지/예측이 가능하다.== 다만 미세한 공정 조건에 대해서는 전통 모델(DLinear, LSTM, WDCNN)이 더 나을 수도 있다.

시계열 전용 모델 대신 새로운 접근 : **LLM의 시계열 데이터 분석**
