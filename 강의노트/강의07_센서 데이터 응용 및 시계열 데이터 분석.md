**ANN**은 전체 데이터를 활용하여 가중치 업데이트함.
**RNN(순환신경망)** 은 전체 가중치는 고정, 순차적으로 관측치를 활용하여 시점 별 정보가 다음 관측치에 반영되도록 은닉층을 계산한다.
	ANN보다 더 많은 parameter가 발생한다.
	직전 data의 은닉층 정보를 다음 data에 반영할 때, 최근의 data가 더 많이 반영된다. ==시퀀스가 길어질수록 과거 정보는 소멸된다.==(기울기 소실 문제)

**LSTM**은 RNN의 단점을 해결하기 위해 등장했다. 
	Long Shor-Term Memory

RNN에는 여러 모델이 있다. 기본 모형은 *'바닐라'* 라 부른다.
	GRU, Bi-directinal LSTM(순/역방향 분석 결과를 mix)가 RNN에 속함

## 2. 센서 데이터 분석
현업에 적용 가능성 높은 내용

**FDC data**를 분석하는 방법들
	1) 다양한 feature 변환 방식(통계량)을 통해 변수들로 변환할 수 있다. 정형 data로 변환하고나면 ML, MLP, LSTM 등으로 모델링 가능하다.
	2) 데이터를 특정 길이로 잘라서 합친 후 2D배열로 변환할 수도 있다. CNN, LSTM으로 모델링 가능하다.
		잘라진 경계의 data는 소실되는 단점이 있다.
	3) 시계열 데이터를 이미지화(인코딩)하면 딥러닝 모델에 적용하여 분류 모형이나 이상 탐지 알고리즘에 적용할 수 있다. CNN, LSTM으로 모델링 가능하다.

**합성곱 신경망**(CNN:Convolution Neural Network)은 인간의 시각 처리 방식을 모방하여 주로 이미지나 영상 데이터를 분석하는 데 사용되는 딥러닝 모델입니다. 입력 데이터의 특징을 스스로 추출하는 합성곱층(Convolutional Layer)과 이미지의 크기를 줄여 연산량을 낮추는 풀링층(Pooling Layer)을 조합하여 객체 인식 및 분류에 뛰어난 성능을 발휘합니다.

## 1. 시계열 데이터분석

### 시계열 정의
- 시간의 경과에 따라 측정된 값, 또는 **순서에 의미가 있는 데이터**
- 예: 주가, 공정 센서, 동영상(이미지 시퀀스), 텍스트, 염기서열 등
- **단변량 / 다변량** 구분 가능

### 1D Convolution
- 시간축 벡터에 필터(Window) 적용, **stride·필터 수** 조절
- 패딩 없으면 출력 길이가 짧아짐

### LTSF-Linear 계열 (Transformer 비판)
- Transformer는 추세·주기성 수치 시계열에서 **의미 정보 부족 → 정보 손실** 논쟁
- **단순 선형·분해 모델**이 순차성 보존하며 추세·주기성 추출에 유리한 경우 많음

| 모델 | 핵심 |
|------|------|
| **Linear** | **단 하나의 선형 레이어**만으로도 Transformer보다 우수할 수 있음. History L → Forecast T |
| **DLinear** | **이동 평균 커널**로 **Trend + Seasonal** 분해 후 각 성분에 선형층 적용·합산. 명확한 추세 있을 때 강함 |
| **NLinear** | 분포 이동 대응. 입력에서 **시퀀스 마지막 값 뺀 뒤** 선형 통과, 예측 전 **다시 더함**(간단 정규화) |

### 전력사용량 예제 (DACON)
- 추세↑ + **일 단위 주기성** 존재
- 비교 결과 **DLinear** Test 성능 최고
- 주기성 가중치상 **최근 timestep(약 60~70)** 이 예측에 중요

## 2. 순환신경망 (RNN)

### 핵심 정의
- 뉴런이 처리 후 재사용하도록 **cycle** 보유
- 수식: $h_t = f_W(h_{t-1}, x_t)$
- 펼치면 $h^{(t)}$는 $h^{(t-1)}$과 $x^{(t)}$로 정의

### ANN vs RNN
- **ANN**: 배치/전체를 독립 샘플처럼 가중치 업데이트
- **RNN**: **가중치 공유(parameter sharing)** + 시점별 은닉층이 다음으로 전달
- 공유 안 하면 **sequence 학습 불가** (파라미터 $U, V, W$ 공유)

### 학습
- **BPTT**: 전체 시퀀스 forward 후 시간 축으로 역전파
- **Truncated BPTT**: chunk 단위 학습. 랜덤 샘플링은 시계열 구조 손상 → **청킹**으로 업데이트

### 장단점
- 장점: 위치/순서에 따라 다르게 처리, 구조 단순
- 단점: 요소 간 관계·통계 활용 약함, **시퀀스 길어질수록 과거 정보 소멸** → **LSTM**으로 해결

## 3. LSTM

### 등장 배경
- 관련 정보와 사용 지점이 멀면 RNN 학습 저하 → **vanishing gradient (기울기 소실)**
- **LSTM (Long Short-Term Memory)** 이 이를 완화

### 구조 핵심
- RNN의 $h_t$에 **cell state $C_t$** 추가
- **Gate**로 셀 스테이트에 정보 **더하거나 제거**
- ResNet **skip connection** / highway와 유사하게 오래된 정보 보존

### 관련 변형
- **GRU**, **Bi-directional LSTM**(순·역방향 결합 → 차원 약 2배)
- 기본 RNN을 **바닐라**라고 부름
- 입력·hidden은 **여러 개** 가능 (단일 노드만 쓰는 구조가 아님)

## 4. 센서 데이터분석

### 특성
- 제조 공정: 온도·압력·유속 등 **다변량·상관 있는 시계열**
- 정상/불량 판별, 고장 원인 식별에 활용
- ML은 DL보다 **셋업이 가벼움**

### 변환 방식 3종
1. **Feature 통계량** → 정형 $X\to Y$ (ML/MLP/LSTM) — 원본 정보 손실 가능
2. 1D를 잘라 **2D 배열 이미지** — 경계 구간 정보 소실 가능
3. **인코딩(이미지화)** 후 CNN/LSTM — 분류·이상탐지

### 주요 인코딩
| 기법 | 요점 |
|------|------|
| **GAF** | 시간 종속성·상관 보존, **일대일 대응 → 시계열 복원 가능** |
| **Spectrogram** | 시간–주파수–강도 시각화, STT에도 활용, 베어링 결함 구분 예 |
| **Scalogram** | **웨이블릿 변환**을 색으로 표현한 시간–주파수 분석 |

### MAE (Masked AutoEncoder)
- 라벨 없이 **자기지도 학습**
- 일부 **masking** 후 복원 학습 → 인코더 출력을 특징 벡터로 사용

### 응용
- 이미지화 후 CNN (다중 채널 유지 시 **센서별 영향 추적** 가능)
- 알람 **조기 레이블링** + Multi-label + CNN → 정확도 대략 80%+ (예지보전)

## 5. WDCNN

### 동기
- GAF 등 변환 → **정보 손실** 가능
- **Raw Vibration Signal**을 직접 CNN 입력

### 구조
- 1층 **Wide Kernel (64×1)**: 고주파 잡음 제거·주요 특징 추출
- 이후 **작은 커널 (3×1)**: 세밀한 결함 특징
- **Batch Normalization**으로 학습 안정·가속

### Domain Shift & AdaBN
- 공장·계절·부하 변화 → **Domain Shift**
- **AdaBN (Adaptive Batch Normalization)**: BN 파라미터만 새 도메인으로 갱신, **전체 재학습 불필요**
- AdaBN 후 심한 잡음(SNR -4dB)에서도 정확도 **90%+** 유지 사례

### 깊이별 표현
- 초기 층: 정상·결함 특징 혼재 → 깊은 층일수록 분리 → FC에서 분류

-----[연습 문제: 빈칸 채우기 및 참/거짓]

### 빈칸 채우기
1. DLinear는 **이동 평균**으로 입력을 (          ) 성분과 나머지(계절성) 성분으로 분해함.  
   → 정답: **Trend (추세)**

2. NLinear는 입력에서 시퀀스의 (          ) 값을 뺀 뒤 선형층에 넣고, 예측 전에 다시 더함.  
   → 정답: **마지막 (last)**

3. RNN이 시퀀스를 학습하려면 시점마다 (          ) 를 해야 함.  
   → 정답: **파라미터(가중치) 공유 (parameter sharing)**

4. RNN의 시간 축 역전파 학습법을 특히 (          ) 라고 함.  
   → 정답: **BPTT** (전체 대신 chunk면 **Truncated BPTT**)

5. LSTM이 RNN에 추가한 핵심 상태는 (          ) 이며, (          ) 로 정보를 더하거나 제거함.  
   → 정답: **Cell State $C_t$** / **Gate**

6. GAF는 일대일 대응이라 이미지에서 다시 (          ) 로 복원 가능함.  
   → 정답: **시계열 데이터**

7. 스케일로그램은 (          ) 변환 값을 2D 색으로 표현한 것임.  
   → 정답: **웨이블릿 (Wavelet)**

8. WDCNN 첫 합성곱은 (          ) 커널로 고주파 잡음을 줄이고, 도메인 변화 대응에는 (          ) 을 씀.  
   → 정답: **Wide (넓은, 예: 64×1)** / **AdaBN**

### 참(O) vs 거짓(X)
1. 텍스트·동영상처럼 절대 시간(타임스탬프)은 없어도, 순서만 의미 있는 데이터도 시계열(시퀀스) 데이터로 다룸.  
   → **O**

2. Transformer는 항상 장기 시계열 예측에서 Linear/DLinear보다 우수함.  
   → **X** (명확한 추세·주기성 수치 시계열에선 단순 선형이 더 나을 수 있음)

3. ANN처럼 시점을 완전 독립 샘플로만 다루면 시계열 흐름 반영이 약해질 수 있음.  
   → **O**

4. Truncated BPTT에서 시점을 무작위로 뽑아 학습해도 시계열 구조는 항상 보존됨.  
   → **X** (랜덤 샘플링은 시계열 정보 손실 → 청킹으로 순서 유지)

5. 시퀀스가 길어져도 vanilla RNN은 과거 정보를 안정적으로 보존함.  
   → **X** (기울기 소실로 과거 정보가 소멸하기 쉬움 → LSTM)

6. Spectrogram은 주로 시간–주파수–강도를 시각화해 파동·결함 특성을 봄.  
   → **O**

7. WDCNN은 반드시 GAF로 이미지화한 뒤 CNN에 넣어야 함.  
   → **X** (원시 진동 신호를 직접 입력하는 것이 핵심)

8. Domain Shift 발생 시 AdaBN은 모델 전체를 처음부터 재학습해야 함.  
   → **X** (BN 파라미터만 새 도메인으로 갱신, 전체 재학습 불필요)

9. Bi-LSTM은 순방향과 역방향 결과를 결합해 표현 차원을 키울 수 있음.  
   → **O**

10. MAE(Masked AutoEncoder)는 라벨이 반드시 있어야만 특징을 학습함.  
   → **X** (마스킹 복원을 통한 **자기지도 학습**, 라벨 없이 가능)
