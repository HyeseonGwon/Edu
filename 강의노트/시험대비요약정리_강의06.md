### **1. CNN(합성곱 신경망)의 등장 배경 및 필요성
 * **기존 방식의 한계**: 과거 MLP나 플래트닝 방식은 이미지를 단순 1차원으로 펼쳐 계산하므로, 사물이 약간만 뒤틀리거나 평행 이동하는 등 형태적 예외상황이 발생하면 완전히 다른 입력으로 인식하여 대응이 불가능했음.
 * **합성곱층의 필요성**: 이미지 전체를 한 번에 연산하지 않고 이미지의 국소적 부분을 나누어 계산함. 이를 통해 시간과 컴퓨터 자원을 크게 절약하면서도 이미지의 세밀한 부분까지 효과적으로 분석할 수 있음.
### **2. 합성곱 연산의 주요 속성 및 핵심 개념
 * **주요 속성**: 이미지의 특정 모서리에서 학습한 패턴을 다른 위치에서도 똑같이 인식할 수 있는 **평행 이동 불변성(Translation Invariant)** 특징을 가짐. 이 덕분에 적은 수의 훈련 샘플로도 우수한 일반화 능력을 확보할 수 있으며, 레이어별로 단순 선에서 추상적 개념으로 발전하는 **공간적 계층 구조(Spatial Hierarchy)** 를 학습함.
 * **핵심 개념**: 국소 영역만을 연결하여 특징을 추출하는 **지역 연결성(Local Connectivity)** 과 하나의 필터(커널) 파라미터를 전체 이미지 영역에 공통 적용하는 **가중치 공유(Weight Sharing)** 가 핵심임. 전통적 신경망에 비해 모델의 학습 변수(가중치) 수를 획기적으로 줄여줌.
 * 스트라이드: 필터를 이동시키는 픽셀 단위. 얼마나 듬성듬성 캡쳐할 것인가.
 * 0-패딩: 출력 이미지의 크기를 입력보다 작아지지 않도록 edge에 dummy(0)를 추가하는 방법.
 * **필터 갯수만큼 feature map이 배로 불어난다.** 3채널(RGB)이더라도 합성곱 과정에서 하나의 층으로 더해지기 때문에, 오로지 필터 갯수만큼 불어난다.
 * 특성 맵은 활성화함수(ReLU:음수는 0으로)로 한번 더 변경됨.
 * 층 쌓기 앞부분에서는 CNN 2차원 공간 구조 파악 
	 * Convolution -> ReLU -> Convolution -> ReLU -> Pooling -> Convolution -> ReLU -> Pooling
 * 층 쌓기 뒷부분에서는 ANN fully conneted 학습. 정답지를 보고 가중치(필터)를 수정한다.
### **3. 풀링(Pooling) 및 ResNet(잔여 학습)
 * **풀링(Pooling)**: 합성곱층을 거친 특성 맵의 채널 개수는 그대로 유지하면서 가로·세로 차원만 다운 샘플링하여 연산량을 감소시킴. 대상 영역에서 가장 큰 값만 추출해 특징을 강조하는 **최대 풀링(Max Pooling)** 이 주로 선호됨.
 * **ResNet**: 층이 극도로 깊어질 때 모델 최적화가 어려워지고 그레디언트 소실/폭발이 일어나는 문제를 해결하기 위해 도입됨. 원본 입력 데이터를 가중치 계층을 건너뛰고 다음 층으로 직접 전달해 주는 **Skip Connection(지름길 연결)** 구조를 사용하여 비선형 함수 뒤에 **항등 매핑(Identity Mapping, $f(x) \rightarrow x + f(x)$)** 을 구현함으로써 깊은 망도 안정적으로 학습시킴.
 * ResNet의 앙상블 효과: 레즈 블록을 $n$개 쌓으면 $2^n$개의 경로가 생긴다. (경로 다양화) 
 * ResNet의 **배치 정규화** 적용: 배치 정규화는 각 컨볼루션 계층 바로 뒤에 적용하여 레이어 입력 분포를 일정하게 해준다. 깊은 신경망에서 훈련을 더욱 용이하게 만들고 **과적합을 방지함**
### **4. Autoencoder(오토인코더)의 구조와 원리
 * **구조**: 입력 데이터를 저차원으로 압축하는 **인코더(Encoder)**, 압축된 유용한 정보의 압축본인 **코드(Code/Features)**, 복원 성능을 검증하며 원본 형태로 되돌리는 **디코더(Decoder)** 로 구성된 비지도 학습 모델임.
 * **이상치 탐지 원리**: 입력 x와 복원 데이터 x' 사이의 오차 제곱인 **재구축 오차(Reconstruction Error, \|x - x'\|^2)** 를 손실함수로 두고 이를 최소화하도록 정상 데이터만을 사용하여 학습함. 이후 비정상(불량) 데이터가 들어오면 압축 후 복원하는 과정에서 제대로 복원되지 못하고 품질이 깨지므로, **재구축 오차가 설정된 임계값보다 크게 발생하는 현상**을 활용해 불량을 잡아냄.
### **5. 임베딩 기반 이미지 이상탐지
 * **개념**: 정상 데이터를 수만 단위 이상 대규모로 구축해야 하는 비용 문제를 극복하기 위해, 대량의 데이터로 이미 학습이 완료된 **사전 훈련된(Pre-trained) 모델(예: ResNet)** 의 고차원 특징 임베딩 공간을 빌려와 활용하는 방식임.
 * **기법 및 한계**: 사전 훈련 모델 추출 벡터와 **kNN(k-Nearest Neighbors)** 알고리즘을 결합하여 거리를 기반으로 이상치 점수를 매김. 단, 불량 데이터끼리 특정 클러스터를 형성하고 있으면 정상으로 오판할 위험이 존재하므로, 이를 보완하기 위해 **가우시안 혼합 모델(GMM)** 이나 데이터 밀도 분포(정상은 밀집, 비정상은 Sparse)를 평가하는 밀도 기반 방식을 접목할 수 있음.
### **6. 주요 Vision AI Task(객체 감지 및 이미지 분할)
 * **객체 감지(Object Detection)**: 이미지 내에서 객체의 정확한 위치 영역을 뜻하는 **Bounding Box**와 해당 객체의 종류(클래스)를 동시에 찾아 출력하는 Task로, 여러 객체를 한 번에 처리하며 고해상도 이미지일수록 탐지 성능에 유리함.
 * **이미지 분할(Segmentation)**: 대략적인 박스 형태를 넘어 **픽셀 단위(Pixel-level)** 로 "어디가 이상(비정상)인가"를 정밀하게 추적하여 마스킹 표시해 주는 Task임. 단순 불량 유무 판정을 넘어 불량이 발생한 정확한 면적과 세부 영역을 시각화해야 하는 반도체·제조 공정 검사 장비 등에 매우 유용하게 활용됨.
### -----[연습 문제: 빈칸 채우기 및 참/거짓]
#### 1. 빈칸 채우기
(1) CNN에서 하나의 필터 가중치 매개변수를 이미지 전체 영역에 똑같이 공통으로 적용함으로써 모델이 학습해야 할 총 변수의 수를 대폭 줄여주는 속성을 ( **가중치 공유 / Weight Sharing** )(이)라고 한다.
(2) ResNet 구조에서 신경망의 깊이가 깊어짐에 따라 최적화가 어려워지고 신호가 왜곡되는 문제를 방지하기 위해, 원본 이미지 데이터를 가중치 계층을 거치지 않고 다음 레이어로 우회하여 그대로 더해주는 경로 연결 구조를 ( **Skip Connection / 지름길 연결** )(이)라고 한다.
(3) 오토인코더 모델 기반의 불량 탐지 시스템에서 원본 입력 데이터와 디코더를 거쳐 최종 출력·복원된 데이터 사이의 차이 값을 뜻하며, 이 수치가 임계치보다 클 때 불량으로 판정하게 만드는 기준 지표는 ( **재구축 오차 / Reconstruction Error** )(이)다.
(4) 객체 감지(Object Detection) Task에서 단순 이미지 분류를 넘어 대상 물체가 위치한 네모난 영역 좌표를 정확히 묶어 인지해 주는 출력 형태를 ( **Bounding Box** )(이)라고 한다.
#### 2. 참(O) vs 거짓(X) 판단
(1) CNN의 합성곱 연산을 통해 추출된 특징 패턴은 평행 이동 불변성(Translation Invariant)을 지니므로 이미지의 오른쪽 아래에서 학습된 패턴이라도 왼쪽 위 모서리에서 발견되었을 때 정상적으로 인식할 수 있다. ( **O** )
(2) 풀링(Pooling) 레이어 연산을 거치면 연산량 감소를 위해 가로·세로의 데이터 차원 수가 절반 등으로 다운 샘플링되며, 이때 특성 맵의 채널 개수 또한 차원 축소 비율과 동일하게 함께 감소한다. ( **X** ) *(설명: 풀링 연산 시 채널 개수는 그대로 유지되며 데이터 개수만 감소함)*
(3) 사전 훈련된 모델과 kNN을 결합한 임베딩 기반 이상탐지 방식은, 비정상 불량 데이터들이 고유한 특성을 지니고 자기들끼리 빽빽하게 클러스터(군집)를 이루고 있을 때 오히려 정상 데이터로 오판할 수 있는 잠재적 한계가 존재한다. ( **O** )
(4) 이미지 분할(Segmentation) 기술은 이미지 내 사물들을 픽셀 단위(Pixel-level)로 쪼개어 경계선과 영역을 표시해 주므로, 정밀하게 장비 이미지의 특정 불량 영역을 추적하고 가공해야 하는 실제 제조 공정 과제에 매우 적합하다. ( **O** )
