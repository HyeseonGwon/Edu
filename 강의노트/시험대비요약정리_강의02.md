## 1. 머신러닝 및 인공신경망 기초

- **머신러닝 정의**: 데이터(X, Y)를 보고 입력과 출력을 연결하는 규칙 **f(학습된 함수)** 를 자동으로 찾아내는 방법.
- **인공신경망(ANN)**: 인간의 신경망 원리를 모방하여 복잡한 데이터 관계를 학습하는 머신러닝 기법.
	- 신경망은 feature를 사람이 설계하지 않는다. 자동 학습 되는 것이다.
	- 은닉층 수가 많아지면 심층신경망이라고 한다.
	- 입출력 구조만 맞추면 모듈화가 쉽다.
- **활성화 함수 (Activation Function)**: 전달 함수라고도 한다. 노드의 가중합을 활성화 함수로 보냄. 사용자가 선택하는 하이퍼파라미터이다.
    - **시그모이드(Sigmoid)**: 출력값을 **0~1 사이의 확률** 형태로 변환.(누운 S)
    - **렐루(ReLU)**: 음수는 0, 양수는 x를 그대로 출력하며 **기울기 소실 문제** 해결에 유리.
    - **소프트맥스(Softmax)**: 출력값의 총합이 항상 **1**이 되도록 정규화하여 주로 출력층에서 사용.
- **비용함수 (Loss Function)**:
    - **MSE (Mean Squared Error)**: 수치 예측(회귀)에서 예측값과 실제값의 차이를 측정.
    - **크로스 엔트로피(Cross Entropy)**: 분류 문제에서 오차를 측정.
- **학습 방법**: **역전파(Backpropagation)** 알고리즘을 통해 오차를 뒤에서 앞으로 전달하며 **가중치(Weight)** 를 조정. 오차를 줄여나가는 방향으로 업데이트한다.(편미분)

학습이 끝난 모델의 가중치들과 은닉층 구조를 알고 있으면, 동일 입력에 동일 출력을 얻을 수 있다. 단, Temperature 설정에 따라 다르게 나올 수 있다.

## 2. 과적합과 일반화
최적화와 일반화는 동일하지 않다!
신경망 학습 방향은 비용함수를 최소화하는 방향이지만, 우리가 원하는 것은 새로운 데이터에도 잘 맞는 일반화 성능이다. (머신러닝은 외사합-주어진 데이터의 범위를 벗어나는 문제-에 약하다)

- **과적합(Overfitting)**: 모델이 학습 데이터에만 과하게 맞춰져 새로운 데이터에 대한 성능이 떨어지는 현상.
#### 과적합 방지 기법: 일반화 성능을 향상시키는 방법
- **더 많은 양질의 데이터 확보**: 양질이란 레이블이 정확한 것이다. 확보 어려운 상황이 더 흔함
- **데이터 증강(Data Augmentation)**: 데이터를 변형하여 학습 양을 늘림. 도메인 지식이 필요
- **조기 종료(Early Stopping)**: 검증 오차가 증가하기 전에 학습을 멈춤.
- **드롭아웃(Dropout)**: 학습 시 무작위로 일부 노드를 제외하여 정규화 효과를 얻음.

## 3. 자연어 처리(NLP)와 임베딩

- **원-핫 인코딩(One-hot Encoding)**: 단어를 0과 1로만 표현. **희소성(Sparse)** 문제와 단어 간 **의미적 유사성**을 반영하지 못하는 한계가 있음.
- #### 임베딩(Embedding)
	데이터를 숫자의 집합(벡터)으로 표현하며, **의미가 비슷한 데이터**는 숫자 공간에서도 가깝게 위치하도록 만듦. 계산이 가능한 형태로 바꾸주는 것이다.
	**임베딩을 활용해 할 수 있는 작업들**
		- 임베딩 벡터 간의 거리 계산
		- 의미 기반 검색
		- 유사한 문서나 문장 추천
		- Vector DB에 저장
		- 텍스트 분류와 군집화에 활용
		- 이상치 탐지
	
- **Word2Vec**: 주변 단어로 중심 단어를 예측하거나 그 반대로 학습하는 **자기지도학습** 구조.

정보를 많이 가지고 있는 개체: 텍스트>이미지>센서 데이터

## 4. 주요 언어 모델 (BERT vs GPT vs T5)

| 모델       | 구조                  | 학습 방식                       | 주요 특징                         |
| -------- | ------------------- | --------------------------- | ----------------------------- |
| **BERT** | **Encoder-only**    | **Denoising (MLM)** Masking | **양방향 문맥** 이해, 분류 및 QA에 강점    |
| **GPT**  | **Decoder-only**    | **Autoregressive**          | **순차적 다음 단어 예측**, 자연스러운 문장 생성 |
| **T5**   | **Encoder-Decoder** | **Text-to-text**            | 모든 문제를 텍스트 변환으로 처리, 범용성       |

### 연습 문제: 빈칸 채우기 및 참/거짓]

**1. 빈칸 채우기**

- (1) 신경망 학습에서 전체 데이터를 한 번 사용하여 가중치를 업데이트하는 단위를 ( **에포크 / Epoch** )(이)라고 한다.
- (2) BERT의 핵심 학습 기법 중 하나로, 문장의 일부 단어를 가리고 주변 문맥으로 이를 예측하는 방식을 ( **Masked Language Model / MLM** )(이)라고 한다.
- (3) 대규모 데이터로 모델의 기본기를 학습시켜 둔 모델을 ( **파운데이션 모델 / Foundation Model** )(이)라고 부른다.

**2. 참(O) vs 거짓(X) 판단**

- (1) 렐루(ReLU) 함수는 시그모이드 함수보다 학습 속도가 빠르고 안정적인 학습이 가능하다. ( **O** )
- (2) 원-핫 인코딩은 단어 간의 의미적 유사도를 계산하기에 매우 적합한 방식이다. ( **X** ) - 단어 구분만 가능하고 의미 파악은 불가능함.
- (3) GPT는 양방향 문맥을 동시에 참고하여 문장을 이해하는 데 최적화된 모델이다. ( **X** ) - GPT는 한 방향(왼쪽→오른쪽)으로 다음 단어를 예측하는 생성 중심 모델임.
