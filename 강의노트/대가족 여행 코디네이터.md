## 프로젝트 개요
- **무엇**: 몸이 불편하거나 매운 것을 못먹는 가족들을 위한 식사/숙소 추천 에이전트.
- **흐름**: 대화로 "누구와 · 무엇을(식당/숙소) · 어디로" 3가지를 모으면 → 후보를 **한 곳씩** 웹검색·검증 → 조건(계단 적음·아이 메뉴 등)을 **모두 통과한 곳만** 지도·리스트에 추천.
- **구조**: "화면(Streamlit)"과 "두뇌(FastAPI + LangGraph)"를 분리, 결과는 실시간으로 하나씩 누적 표시.

#### 주된 특징 : 
여러 개의 키워드를 주고 웹사이트 검색을 하는 것이라 정확도 기반으로 top-k만 가져오는 방식은 너무 느림... 속도를 위해 1건씩 검증, 조건 충족시 바로 스트리밍되게 함.

## 사용된 Tech stack
| 구분        | 기술              | 역할(층)                                             |
| --------- | --------------- | ------------------------------------------------- |
| 화면        | **Streamlit**   | 프런트엔드 — 사용자가 보는 화면(대화창·지도·리스트)                    |
| ^^        | **Folium**      | 지도 시각화 — 추천지를 지도 위 핀·원으로 렌더링                      |
| 로직        | **FastAPI**     | 백엔드 — 로직을 HTTP API로 노출(`/chat`, `/search/step` 등) |
| ^^        | **LangGraph**   | 에이전트 — 대화 상태 관리·조건 수집 워크플로우                       |
| 데이터·외부 연동 | **Kakao Local** | 지오코딩(정밀) — 상호명 → 실제 좌표                            |
| ^^        | **Nominatim**   | 지오코딩(폴백) — 검색 중심 좌표 및 Kakao 실패 시 대체 좌표            |
| ^^        | **DuckDuckGo**  | 웹 검색 — 후보 발굴·리뷰 스니펫 수집                            |
## **검색 신뢰도** 향상 과정
| # | 문제 상황 | 개선 내용 |
|---|---|---|
| 1 | 추천 후보에 **식당명이 아니라 '메뉴 이름'** 이 잡히는 경우 발생 | 식당명과 메뉴명을 **구분**하도록 추출 프롬프트 강화 |
| 2 | 조건 부합 **근거를 하이퍼링크**로 제공하려 했으나, 검색 최상단 사이트를 연결해 **엉뚱한 페이지로 이동**(오판) | LLM이 실제 판단 근거로 삼은 결과의 **실제 URL을 매핑**하도록 개선 |
| 3 | 식당의 **정확한 좌표를 몰라** 지도 위 임의 위치에 핀이 찍힘 | **카카오맵 무료 API**를 발급·연동해 실제 좌표를 반영 |

## **사용자 편의성** 개선 과정
| #   | 문제 상황                                              | 개선 내용                                                              |
| --- | -------------------------------------------------- | ------------------------------------------------------------------ |
| 4   | (정확한 좌표 확보 후) 지역 결과가 적을 때 **범위를 넓혀 다시 찾을 수단**이 없음  | **검색 반경 슬라이더 + '다시 찾기' 버튼**을 추가해 사용자가 반경을 조정하며 재검색                 |
| 5   | LLM이 **근거 없이 검색 조건을 채우면 오작동**, 반대로 너무 엄격하면 대화가 경직됨 | 분명한 의도는 추론해 채우고(배고프다→식당, 떠나고싶다→숙소), **장소(지역)는 명시적 답변을 유도**         |
| 6   | 식당 검색시 특정 메뉴가 먹고싶을 수 있음                            | 식당을 검색하는 경우 **선택적으로 '메뉴' 요소도 검색**하고, '메뉴 일치' 뱃지를 붙여 리스트 상위로 올라오게 함 |
